### 📝 실험개요
Paired dataset 기반으로 학습하는 모델인 Pix2Pix와 Unpaired dataset 기반으로 학습하는 CycleGAN을 활용한 여러가지 실험을 통해 각 모델의 장점과 단점을 분석하는 것을 목표로 실행하였습니다. 이번 연구에서 진행한 실험은 3가지로 다음과 같다
- Colorization
- Cartoonization
- Denoising
